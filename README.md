# Annotation Refiner 项目说明

## 项目概述

Annotation Refiner 是一个用于YOLO标注数据清理和优化的工具，帮助用户更高效地管理和修正标注框。支持自动清理重复标注、嵌套标注和相似标注，并提供丰富的交互功能。

## 安装说明

### 1. 环境要求
- Python 3.8+
- 依赖库：nicegui, numpy, opencv-python

### 2. 安装步骤
```bash
# 克隆项目
git clone https://github.com/guenqing/LableOpt.git

# 进入项目目录
cd LableOpt

# 安装依赖
python -m pip install -r requirements.txt

# 启动应用
python anno_refiner_app/main.py
```

### 3. 访问应用
启动后，在浏览器中访问显示的地址（默认 http://localhost:8080）

## 功能特性

### 核心功能
1. **标注框清理**：自动检测和删除重复标注、大框框小框、不同类别的相似标注
2. **交互式标注**：支持手动编辑标注框、调整大小和位置
3. **视野保持**：切换图片时智能保持视野状态
4. **标签显示优化**：智能调整标签位置，避免重叠
5. **Box List**：显示所有标注框信息，包括尺寸和类别
6. **嵌套框选择**：点击嵌套标注框时自动选择最小的标注框

## 使用指南

### 1. 首页配置
在首页填写以下信息：
- **Images Path**：图片文件夹路径
- **GT Labels Path**：GT标签文件夹路径
- **Pred Labels Path**：预测标签文件夹路径（可选）
- **Output Path**：输出文件夹路径

### 2. 开始分析
点击 **"Analyze"** 按钮分析数据：
- 系统会自动扫描文件夹，查找图片和对应的标注文件
- 计算 **Pending Samples**（待处理样本数）
- Pending Samples = 同时存在图片、GT标签和预测标签且未处理的样本数量

### 3. 进入标注
点击 **"Start Annotation"** 进入标注页面

### 4. 基本操作
- 使用 **左右箭头按钮** 或 **键盘左右键** 切换图片
- 点击标注框进行选中
- 拖动标注框调整位置
- 拖动控点调整大小
- 使用快捷键快速切换类别

### 5. 保存标注
- 点击 **"Save"** 保存当前图片的标注
- 点击 **"Save All"** 保存所有修改

## 高级功能

### Clean Annotations（标注清理）
自动清理以下类型的标注框：

1. **重复标注**：两个标注框IoU超过阈值时，保留较大的
2. **大框框小框**：大框完全包含小框时，删除大框
3. **相似标注**：不同类别但位置高度相似的标注框，两个都删除

**使用方法**：
1. 勾选 **"Clean Annotations"** 复选框
2. 在下方选择阈值：**Low** / **Medium** / **High**
3. 系统会自动处理当前图片
4. 在图片左上角显示清理结果（如 "Clean Annotations：3"）
5. 取消勾选可撤销清理

**阈值说明**：
- **Low**：较宽松的标准，清理较少
- **Medium**（默认）：平衡标准
- **High**：较严格的标准，清理较多

### 视野保持功能
- 当从有标注框的图片切换到没有标注框的图片时，保持前一图片的视野（缩放和位置）
- 当切换回有标注框的图片时，自动执行auto focus显示所有标注框

### Box List面板
右侧显示当前图片的所有标注框：
- 显示类型（GT/Pred）、序号、类别、尺寸
- 选中状态同步
- 点击可快速选中对应标注框

## 技术说明

### 标注框清理逻辑
1. **重复标注**：IoU > 阈值时，保留较大的标注框
2. **大框框小框**：大框包含小框时，删除大框（不考虑类别）
3. **相似标注**：不同类别且IoU > 阈值且边界相似（至少三边差值 ≤ 10px）时，删除两个标注框

### 标签显示逻辑
- 尝试在标注框的6个位置显示标签：左上角、右上角、左下角、右下角、左侧中间、右侧中间
- 自动选择第一个不重叠且在图片边界内的位置
- 标签显示类别ID，紧挨着标注框

### 视野保持逻辑
- 切换到有标注框的图片时：保存视野状态 + 执行auto focus
- 切换到无标注框的图片时：恢复前一图片的视野状态

## 常见问题

### Pending Samples为0
- **原因**：同时存在图片、GT标签和预测标签且未处理的样本数量为0
- **解决**：
  1. 检查文件路径是否正确
  2. 确保图片和标注文件命名对应
  3. 检查GT标签文件夹中是否有标注文件
  4. 确保输出文件夹中没有已处理的文件

### 标注框选择困难
- **原因**：嵌套标注框总是选中大的
- **解决**：系统已优化，点击时会自动选择最小的标注框

### 标签显示重叠
- **原因**：多个标注框位置靠近
- **解决**：系统会自动调整标签位置，尝试不同的位置避免重叠

### 快捷键不生效
- **原因**：焦点在输入框等元素上
- **解决**：点击图片区域后再使用快捷键
