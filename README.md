# Annotation Refiner 项目说明

## 项目概述

Annotation Refiner 是一个用于YOLO标注数据清理和优化的工具，帮助用户更高效地管理和修正标注框。

## 安装说明

### 1. 环境要求
- Python 3.8+
- 依赖库：nicegui, numpy, cleanlab, opencv-python

### 2. 安装步骤
```bash
# 安装依赖
python -m pip install -r requirements.txt

# 启动应用
python anno_refiner_app/main.py
```

## 功能特性

### 核心功能
1. **标注框清理**：自动检测和删除重复标注、大框框小框、不同类别的相似标注
2. **交互式标注**：支持手动编辑标注框、调整大小和位置
3. **视野保持**：切换图片时保持视野状态，提升标注效率
4. **标签显示优化**：智能调整标签位置，避免重叠
5. **Box List**：显示所有标注框信息，包括尺寸和类别

### 新增功能

#### 2026-03-06 更新
- **视野保持**：当切换到没有标注框的图片时，保持前一图片的视野状态
- **标注框选择优化**：点击嵌套标注框时，自动选择最小的标注框
- **标签显示优化**：标签位置智能调整，保证标签挨着标注框
- **控点优化**：标注框控点大小调整为1.8px，避免遮挡画面
- **Box List增强**：显示标注框尺寸信息

#### 2026-03-05 更新
- **Clean Annotations**：自动清理重复标注、大框框小框、相似标注
- **阈值设置**：提供Low/Medium/High三个档次的清理阈值
- **撤销功能**：仅恢复被Clean Annotations删除的标注框
- **状态显示**：在图片左上角显示清理结果

## 使用指南

### 基本操作
1. **加载数据**：在首页输入图片路径、GT标签路径和预测标签路径
2. **开始分析**：点击"Analyze"按钮分析数据，计算Pending Samples
3. **进入标注**：点击"Start Annotation"进入标注页面
4. **导航**：使用左右箭头按钮切换图片
5. **编辑标注**：点击标注框进行编辑，拖动控点调整大小

### 高级功能

#### Clean Annotations
1. **启用**：勾选"Clean Annotations"复选框
2. **阈值设置**：选择Low/Medium/High阈值
3. **效果**：自动清理符合条件的标注框
4. **撤销**：取消勾选后会恢复被清理的标注框

#### 快捷键
- **Ctrl+Z/Y**：撤销/重做
- **= / -**：放大/缩小
- **0**：重置缩放
- **Tab**：循环选择标注框

## 技术说明

### 标注框清理逻辑
1. **重复标注**：IoU > 阈值时，保留较大的标注框
2. **大框框小框**：大框包含小框时，删除大框
3. **相似标注**：不同类别且IoU > 阈值且边界相似时，删除两个标注框

### 视野保持逻辑
- 当切换到有标注框的图片时，保存视野状态并执行auto focus
- 当切换到没有标注框的图片时，恢复前一图片的视野状态

### 标签显示逻辑
- 尝试在标注框的左上角、右上角、左下角、右下角、左侧中间、右侧中间显示标签
- 自动选择不重叠且在图片边界内的位置

## 常见问题

### Pending Samples为0
- 原因：同时存在图片、GT标签和预测标签且未处理的样本数量为0
- 解决：检查文件路径和命名是否匹配，确保GT标签、预测标签和图片文件对应

### 标注框选择困难
- 解决：点击嵌套标注框时，系统会自动选择最小的标注框

### 标签显示重叠
- 解决：系统会自动调整标签位置，尝试不同的位置避免重叠
