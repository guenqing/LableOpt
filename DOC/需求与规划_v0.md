## 需求

我希望针对Object detection标签修正（GT 漏标/ GT 类别错误 /  GT 框不准）工作，设计一个易于操作的集成UI。我将使用训练好的ultralytics YOLO对图片数据进行预测，这些图片数据有对应的标注文件（若无标注则表明目前认为它是负样本）。我希望做这样的事情：
1. 在linux服务器上启动一个服务，使其可以通过端口将一个网页UI传递到本地浏览器，加以使用；
2. 这个UI，初始在A页面，通过左侧三个文本框分别提供图片数据路径，预测标注路径和GT标注路径（YOLO风格），左侧下方有一个按钮"run cleanlab"，按钮下还有一个勾选框"backup annotation folder"，若勾选了，则在后续处理之前先将原预测标注路径和GT标注路径各复制出一个同级的*_bk目录。无论是否备份，所有格式转换以及后续的标注修改，均可以在预测标注路径/GT标注路径下直接实施。
3. 点击"run cleanlab"后，首先对标注文件完成cleanlab（https://github.com/cleanlab/cleanlab）所需的格式转换。然后，使用cleanlab的方法，在A页面右侧分三列展示overlooked, swapped class, badly located的topK个样本。K默认为10，可以通过三列上方的一个"topK"文本框输入自定义数值（允许1~1000）。
4. 确认过三列问题的大致情况后，在三列下方横向排布三个勾选框"include topK overlooked", "include topK class-swapped", "include topK badly-located"，三个勾选框下方有一个"go to annotation tool"。选中1-3个框后，取目标错误类别样本的并集，作为待处理样本，进入页面B。
5. 页面B基本上就是一个常规的标注UI（参考labelImg），通过"display pred"和"display GT"两个勾选框，决定展示问题样本的预测标注框和/或GT标注框（若都不勾选则都不展示）。相比于labelImg稍作简化：由于样本已确定，不需要再选择图像或标签路径；默认展示GT；不要菜单栏，默认YOLO格式；保留"previous"/"next"按键；设置一个"auto save"勾选框。若"auto save"为true，则在进入下一张图片时，将上一张图片的标注结果（即便没有任何改动）保存为*_tmp.txt临时文件（若为负样本则该临时文件为空）。另外，为方便可视化，设置两个勾选框："highlight box border"，若为true，则对标注框展示较粗的边框，否则不展示边框；"highlight box content"，若为true，则用较不透明的颜色展示填充标注框，否则不对标注框填充颜色"。
6. 页面B的右下角设置一个醒目的按键"go back analysis"，点击时，弹窗询问"keep changed as confirmed?"，若点击"yes"，则用*_tmp.txt覆盖对应的原GT，否则删除*_tmp.txt。注意，在完成全部问题样本的标注更新后，通过点击"go back analysis"->"yes"就可以保存这一轮标注结果，然后再进行一次分析。


## 规划

这是一个非常清晰且具有挑战性的需求。为了满足**稳健性、可拓展性、美观性**的要求，同时考虑到"缺乏全栈经验"但精通 Python/CV 的背景，采用**"重后端逻辑，轻前端语法"**的开发路线。

**技术选型**：
- **后端 (Backend):** FastAPI。负责业务逻辑、Cleanlab 计算、文件 I/O。
- **前端 (Frontend):** **NiceGUI**。这是一个 Python 库，它在内部运行 FastAPI，并将 UI 渲染为 Vue.js + Quasar (Material Design)。它支持原生的 Python 异步操作、WebSocket 双向通信（这对"点击保存 -> 立即更新"非常重要），且界面默认就是现代且美观的。
- **数据层 (Data):** **SQLite + SQLModel** (ORM)。使用轻量级数据库记录每一次"分析 Session"的状态、TopK 列表、图片的审阅状态（Pending/Fixed/Skipped）。
- **标注组件 (Annotation):** 基于 SVG 的自定义交互组件（NiceGUI 支持嵌入 JS 或直接操作 SVG 元素），实现完整版 LabelImg 体验（支持选中框高亮、拖拽移动、调整大小）。

### 阶段性开发路线图

#### 阶段一：数据层与核心算法封装 (Python Base)

**目标**：不碰 UI，先写出坚固的 Python 类库，处理所有脏活累活。

1. **定义数据模型** (`src/models.py`)：
   - 使用 SQLModel 定义数据库结构：
     - `AnalysisSession`: 记录一次分析任务（对应页面 A 的一次 Run），包含三个路径、创建时间、top_k 等。
     - `IssueItem`: 记录 Cleanlab 发现的问题样本，包含 session_id、image_name、issue_type (overlooked/swapped/badloc)、cleanlab_score、status (pending/fixed/skipped) 等。
   - 定义 Pydantic 数据模型：
     - `BBox` (xyxy, class_id)
     - `ImageMetadata`
     - 确保 YOLO 格式 (归一化 center_x, center_y) 与 UI 格式 (像素 x1, y1) 的双向转换函数经过了单元测试。

2. **封装 Cleanlab 处理器** (`src/core/analyzer.py`)：
   - 编写一个类 `CleanlabAnalyzer`。
   - 输入：图片/标签目录。
   - 动作：加载数据 -> 转换格式 -> 运行 `cleanlab.object_detection.rank` -> 返回结构化的 TopK 列表。
   - *关键点*：Cleanlab 运行慢，这里预留回调接口（Callback）以便后续对接进度条。

3. **封装文件操作管理器** (`src/core/file_manager.py`)：
   - 实现 `backup_folders()`：备份预测标注路径和GT标注路径到同级*_bk目录。
   - 实现 `save_tmp_annotation()`：保存标注结果为*_tmp.txt临时文件。
   - 实现 `confirm_batch_changes()`：用*_tmp.txt覆盖对应的原GT，或删除*_tmp.txt。

4. **YOLO 工具函数** (`src/core/yolo_utils.py`)：
   - 实现 YOLO 格式 (归一化 center_x, center_y, width, height) 与像素坐标 (x1, y1, x2, y2) 的双向转换。
   - 实现标注文件的读取和写入。

   - 初始化 SQLite 数据库连接。

   - 封装 CRUD 操作：
     - 创建/查询 `AnalysisSession`
     - 批量插入/查询/更新 `IssueItem`

2. **编写 API 接口** (在 `main.py` 或单独的 API 模块中)：
   - `POST /analyze`: 启动分析（后台任务），创建 Session，运行 Cleanlab，将结果存入数据库。

   - `POST /confirm`: 确认批量更改（将*_tmp.txt覆盖原GT或删除）。

#### 阶段四：核心攻坚——标注组件 (Interactive Canvas)

**目标**：在浏览器里画框，且体验流畅。这是最难的一步。

**技术选型**：在 NiceGUI 中使用 `ui.interactive_image`，它允许在图片上叠加 SVG 内容。

**核心逻辑：状态机**
需要处理三种鼠标状态：
1. **IDLE**: 鼠标移动时，检测是否悬停在某个框的边缘（准备调整大小）或内部（准备移动）。
2. **DRAWING**: 按下鼠标左键，且未选中任何现有框 -> 创建新框。
3. **EDITING**: 选中了某个框 -> 拖拽移动或拖拽边角改变大小。

**实现要点** (`src/ui/components.py`)：
- **显示**：将 GT 框和预测框渲染为 `<rect>` 元素（带颜色填充和边框），根据"highlight box border"和"highlight box content"勾选框控制显示样式。
- **交互**：通过 `on_mouse` 事件监听鼠标的 Down/Move/Up，计算坐标差，实时更新 SVG `<rect>` 的属性。
- **选中框**：点击框时高亮显示（边框变色、变粗，填充半透明颜色）。
- **移动框**：选中框后拖拽可移动整个框。
- **调整大小**：检测鼠标是否在框的边缘，拖拽边缘可调整框的大小。
- **删除框**：支持删除选中的框（可通过键盘 Delete 键或右键菜单）。

以下是一个**可直接运行的组件原型代码**（简化版，实现了画框、选中、移动）。可以将其放入 `src/ui/components.py` 并扩充：

```python
from nicegui import ui, events
from typing import List, Optional, Callable
import uuid

class BBox:
    def __init__(self, x: float, y: float, w: float, h: float, label: str = "0"):
        self.id = str(uuid.uuid4())
        self.x = x
        self.y = y
        self.w = w
        self.h = h
        self.label = label
        self.selected = False

    @property
    def svg_str(self):
        # 选中时边框变色、变粗
        color = "red" if self.selected else "yellow"
        stroke_width = 3 if self.selected else 2
        # 填充半透明颜色
        fill_opacity = 0.2 if self.selected else 0.0
        return f'<rect x="{self.x}" y="{self.y}" width="{self.w}" height="{self.h}" ' \
               f'stroke="{color}" stroke-width="{stroke_width}" fill="{color}" fill-opacity="{fill_opacity}" />'

class InteractiveAnnotator:
    def __init__(self, image_source: str, on_change: Callable):
        self.bboxes: List[BBox] = []
        self.active_box: Optional[BBox] = None  # 当前正在画或正在改的框
        self.start_x = 0
        self.start_y = 0
        self.mode = "IDLE"  # IDLE, DRAWING, MOVING
        self.on_change = on_change # 数据回调

        # 核心组件
        self.image = ui.interactive_image(
            image_source,
            on_mouse=self.handle_mouse,
            events=['mousedown', 'mouseup', 'mousemove'],
            cross=True # 显示十字准星
        ).classes('w-full') # 宽度自适应

        self.update_content()

    def update_content(self):
        # 将所有 BBox 转换为 SVG 字符串叠加在图片上
        svg_content = ''.join([b.svg_str for b in self.bboxes])
        self.image.content = f'<svg viewBox="0 0 {self.image.width} {self.image.height}">{svg_content}</svg>'

    def handle_mouse(self, e: events.MouseEvent):
        # 注意：e.image_x 和 e.image_y 是相对于图片像素的坐标
        
        if e.type == 'mousedown':
            # 1. 简单逻辑：点击先检查是否点中现有框（倒序遍历，优先选上层的）
            clicked_box = None
            for box in reversed(self.bboxes):
                if box.x <= e.image_x <= box.x + box.w and box.y <= e.image_y <= box.y + box.h:
                    clicked_box = box
                    break
            
            if clicked_box:
                # 选中模式
                self.deselect_all()
                clicked_box.selected = True
                self.mode = "MOVING" # 简化：暂时只做移动，不做Resize
                self.active_box = clicked_box
                self.start_x = e.image_x
                self.start_y = e.image_y
            else:
                # 画新框模式
                self.deselect_all()
                self.mode = "DRAWING"
                self.start_x = e.image_x
                self.start_y = e.image_y
                # 创建一个初始大小为0的新框
                new_box = BBox(e.image_x, e.image_y, 0, 0)
                self.bboxes.append(new_box)
                self.active_box = new_box

        elif e.type == 'mousemove':
            if self.mode == "DRAWING" and self.active_box:
                # 更新宽和高
                self.active_box.w = e.image_x - self.start_x
                self.active_box.h = e.image_y - self.start_y
            
            elif self.mode == "MOVING" and self.active_box:
                # 计算位移
                dx = e.image_x - self.start_x
                dy = e.image_y - self.start_y
                self.active_box.x += dx
                self.active_box.y += dy
                self.start_x = e.image_x
                self.start_y = e.image_y

        elif e.type == 'mouseup':
            # 修正负宽高（如果从右下往左上画）
            if self.active_box:
                if self.active_box.w < 0:
                    self.active_box.x += self.active_box.w
                    self.active_box.w = abs(self.active_box.w)
                if self.active_box.h < 0:
                    self.active_box.y += self.active_box.h
                    self.active_box.h = abs(self.active_box.h)
                
                # 过滤太小的误触框
                if self.active_box.w < 5 or self.active_box.h < 5:
                    if self.active_box in self.bboxes:
                        self.bboxes.remove(self.active_box)
            
            self.mode = "IDLE"
            self.active_box = None
            self.on_change(self.bboxes) # 触发外部保存/更新

        self.update_content()

    def deselect_all(self):
        for b in self.bboxes:
            b.selected = False
            
    def load_bboxes(self, bboxes: List[BBox]):
        """供外部调用：加载 GT"""
        self.bboxes = bboxes
        self.update_content()

```
